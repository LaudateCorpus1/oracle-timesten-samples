Sample Programs For Developers

Oracle TimesTen and the Application-Tier Database Cache products (TimesTen database from here on) offer developers a number of database interfaces for application development. The interfaces are grouped into three categories depending on your programming language preference.

Java

 For Java developers, you can work with the TimesTen database via:
  • SQL programming using JDBC APIs
         See the JDBC Sample Programs section for more information.

  • EJB APIs with your Java EE application server
    •      See the Java EE Application Server Sample Application section for more information.

  • Object Relational mapping tools such as Hibernate, EclipseLink or OpenJPA
         See the Object Relational Mapping Sample Application section for more information.

C

 For C developers, your API choices include:
  • Standard ODBC
         See the ODBC Sample Programs section for more information.

  • Oracle Call Interface (OCI)
          See the Oracle Call Interface (OCI) Sample Programs section for more information.

  • Pro*C embedded SQL
         See the Pro*C Sample Programs section for more information.

C++

 For C++ developers, the easiest way to program against the TimesTen database is to use the TimesTen TTClasses interface, a C++ library shipped with the product. All of the TimesTen C APIs via the ODBC interface are also available for the C++ programs using TTClasses.

See the TTClasses Sample Programs section for more information.

PL/SQL

 For PL/SQL developers, a number of sample scripts and procedures are provided to show how to use PL/SQL with the TimesTen database. PL/SQL packages, procedures, and functions can be invoked from any of the above supported interfaces, JDBC, ODBC, OCI, Pro*C and TTClasses (C++).

See the PL/SQL Sample Procedures section for more information.
