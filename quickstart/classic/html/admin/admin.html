Configuration and Setup

This area provides tutorials on how to properly install and manage TimesTen instance, create and configure TimesTen databases, user accounts and privileges, start and stop TimesTen database processes, and other database operations. After configuring and creating a TimesTen database, you will learn how to use the TimesTen database as an Application-Tier Database Cache for caching Oracle database tables. You will learn how to configure a cache with different types of cache groups (read, read/write, pre-loaded, loaded on demand, etc.). You will also learn how to configure TimesTen Replication to achieve high availability in the application tier, and how to configure your TimesTen Cache database with a Java EE Application Server and OR mapping tools. Let's get started:

1. Getting Started with TimesTen Installation
  • Introductory section describing the TimesTen installation and instance management. You should start here.

2. TimesTen Database Configuration and Setup
  • TimesTen Database section describing the common TimesTen database configuration and set up.

3. Configuring TimesTen Application-Tier Database Cache
  • TimesTen Application-Tier Database Cache (TimesTen Cache) provides the ability to cache Oracle database tables in the application tier to improve an application's response time. The tutorials in this section consist of step by step examples to set up your TimesTen cache groups and required user accounts and privileges.

4. Configuring Replication between TimesTen Databases
  • Replication provides high availability for TimesTen databases. Replication between TimesTen databases can be achieved with or without Oracle Clusterware. The tutorials in this section describe how to configure and set up replication between TimesTen databases with and without Oracle Clusterware.

5. Configuring Java EE Application Servers to work with TimesTen
  • TimesTen is often used with Java EE application servers. Learn how to configure your application server of choice to use TimesTen Database in the application tier.
