Tools and Utilities

The TimesTen Database provides many tools for database operations. This area contains how-to tutorials for some of these commonly used utilities and built-in procedures.

1. Data Loading
  • This section introduces four TimesTen data loading tools and built-in procedures: ttSizeOra,ttBulkCp, ttLoadFromOracle and ttImportFromOracle.

2. Index Advisor
  • This tool evaluates a given SQL workload and recommends optimal indexes to improve the performance of the associated queries.

3. Utilities and Built-ins
  • This section describes some of the most commonly used TimesTen built-in procedures that are callable programmatically from applications, and a set of utilities for operating and managing the TimesTen databases.

4. Using TimesTen with Oracle SQL Developer
  • Learn how to use the SQL Developer product with your TimesTen database.

5. Using TimesTen with Oracle Enterprise Manager
  • Learn how to use the TimesTen Plugin for Enterprise Manager to monitor and manage the TimesTen database.
